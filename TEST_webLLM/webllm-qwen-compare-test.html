WebLLM browser test — two-model compare. v 1.1 Jonathan Rothberg 2026

Standalone harness for @mlc-ai/web-llm (WebGPU). Defaults use the closest official prebuilt Qwen3 q4-style weights in the current catalog: Qwen3-0.6B-q4f16_1-MLC vs Qwen3-1.7B-q4f16_1-MLC (nearest sizes to “~0.8B q4” and “~2B” until Qwen3.5 prebuilts appear). Each field is a dropdown of all prebuilt model_ids: local disk entries (in webllm-assets/) appear first and bold; the rest use the network on first load. Large models may fail to load on limited GPU RAM — errors are caught and shown without crashing the page. On each load the page checks for webllm-assets/<model_id>/resolve/main/mlc-chat-config.json on this origin (fast when you serve the repo locally). If missing — e.g. on GitHub Pages — weights come from the network. Status lines and each panel show which path was used.
